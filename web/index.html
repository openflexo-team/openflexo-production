We will come back as soon as possible

The Openflexo company stopped its activity in April 2018. IMT Atlantique PASS research team and an association of openflexo users are working on bringing back all the services that were available on this address:
  • The web site with the documentation
  • The artifactory for the software packages
  • The JIRA for all openflexo related bugs and enhancement requests
  • The jenkins and the prepackaged versions of the software
  • The email addresses openflexo.org
During the downtime, the source code is still available at https://github.com/openflexo-team/.


Thank you for your patience!
